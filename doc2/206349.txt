'''Kereta api Senja Utama YK''' ([[Hanacaraka]]: {{jav|}}, ''Sepur Senja Utama Ngayogyakarta'') merupakan [[kereta api bisnis|kereta kelas bisnis]] yang dioperasikan oleh [[PT Kereta Api Indonesia]] di [[Jawa]] dengan relasi [[Stasiun Yogyakarta|Yogyakarta]]-[[Stasiun Pasar Senen|Pasar Senen]], pp. Dinamakan Senja Utama Yogya karena kereta ini berangkat meninggalkan [[Pasar Senen]] ataupun [[Yogyakarta]] pada malam hari.

Kereta ini berhenti beberapa stasiun besar, antara lain, [[Stasiun Wates|Wates]], [[Stasiun Gombong|Gombong]], [[Stasiun Purwokerto|Purwokerto]], [[Stasiun Cirebon|Cirebon]], dan [[Stasiun Jatinegara|Jatinegara]].

Susunan normal rangkaian kereta api ini adalah sebuah lokomotif, tujuh gerbong bisnis (K2), satu gerbong makan dan pembangkit (KMP2/MP3), dan satu gerbong bagasi (B).

== Insiden ==
* Pada tanggal [[13 November]] [[2015]], terjadi kecelakaan yang melibatkan kereta api Senja Utama Yogya dengan truk air minum yang menyeberang perlintasan sebidang di km 29,5 antara [[stasiun Bekasi|Bekasi]]-[[stasiun Tambun|Tambun]]. Akibatnya, lokomotif CC201 83 26 rusak dan tidak bisa bergerak. Tiada korban jiwa.<ref>[http://news.metrotvnews.com/read/2015/11/14/190689/ka-senja-utama-yogyakarta-dan-truk-terlibat-kecelakaan- MetroTV: KA Senja Utama Yogyakarta dan Truk Terlibat Kecelakaan]</ref>
