'''Kereta api Seminung''' adalah kereta api [[KRD]] yang beroperasi di Lampung dengan rute [[Stasiun Tanjungkarang]]-[[Stasiun Kotabumi]]. Kereta api ini hampir berhenti di setiap stasiun yang dilewati. Setiap harinya kereta api ini berjalan 2X sehari PP. Di rute yang sama terdapat [[kereta api Way Umpu]] dan berangkat pagi. Kereta api ini menggunakan jenis [[KRD]] buatan [[INKA]] yang merupakan bantuan dari [[Kementerian Perhubungan]] untuk menggantikan KRD Ruwa Jurai yang kini dalam status afkir.

== Rincian Kereta==

Kereta ini berangkat hanya sekali sehari dan berhenti di stasiun-stasiun sepanjang rute Stasiun Tanjungkarang-Stasiun Kotabumi, dengan set stamformasi sebagai berikut:
# K3 3 09 09 (MEC1 berkabin masinis)
# K3 3 09 10 (T)	
# K3 3 09 11 (T)	
# K3 3 09 12 (MEC2 berkabin masinis)
