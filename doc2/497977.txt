'''Kereta api Fajar Utama Semarang''' adalah [[kereta api bisnis|kereta kelas bisnis]] yang dioperasikan oleh [[PT Kereta Api Indonesia]] di [[Jawa]] yang melayani rute [[Semarang Tawang]]- [[Pasar Senen]]. Dinamakan Fajar Utama Semarang karena berangkat dari [[Stasiun Semarang Tawang|Semarang Tawang]] atau [[Stasiun Pasar Senen|Pasar Senen]] pada pagi hari. Kereta ini melewati beberapa stasiun besar antara lain, [[Stasiun Pekalongan]], [[Stasiun Cirebon]], dan [[Stasiun Jatibarang]]. 

Mulai 19 Agustus 2014, KA ini bersama dengan [[Kereta api Senja Utama Semarang|KA Senja Utama Semarang]] resmi dihentikan operasionalnya dan tugasnya untuk melayani rute Semarang Tawang-Pasar Senen digantikan sepenuhnya dengan [[kereta api Menoreh]].
