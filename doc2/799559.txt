Tabrakan kereta api Juanda 2015
'''Tabrakan kereta api Juanda 2015''' adalah sebuah kecelakaan yang menimpa sebuah kereta api dengan nomor KA 1156 yang menabrak kereta api di depannya dengan nomor KA 1154 pada pukul 15.30-16.00 [[Waktu Indonesia Barat|WIB]] pada tanggal [[23 September]] [[2015]] di lintas [[Bogor]]-[[Jakarta]] tak jauh dari [[Stasiun Juanda]], [[Jakarta Pusat]], [[Indonesia]].<ref>http://megapolitan.kompas.com/read/2015/09/23/16023671/2.KRL.Berbenturan.di.Stasiun.Juanda</ref> Akibat dari peristiwa tersebut, 35 orang luka-luka yang sebagian besar perempuan. Tidak ada korban tewas dalam kecelakaan ini.<ref>http://metro.news.viva.co.id/news/read/678175-dua-krl-tubrukan--35-terluka-sebagian-besar-perempuan</ref>

Akibat kecelakaan ini, 2 set KRL JR 205 terancam berhenti beroperasi. Namun, akhirnya kedua set kereta tersebut digabung, dengan menggunakan kereta-kereta yang selamat. Diduga kecelakaan ini terjadi karena kelalaian masinis KA 1156 yang tidak melihat sinyal masuk sehingga menabrak kereta api didepannya.<ref>http://m.metrotvnews.com/read/2015/09/25/434533/ini-dugaan-sementara-penyebab-tabrakan-krl-di-stasiun-juanda</ref>
