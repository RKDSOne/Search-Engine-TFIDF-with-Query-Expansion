'''Stasiun Soreang''' ('''???''') adalah stasiun kereta api yang akan dibangun di [[Kota Parepare]]. Pada tahap awal, stasiun ini akan terhubung dengan jalur kereta api menuju [[Kota Makassar]]. Namun, rencananya [[jalur kereta api Trans-Sulawesi|jalur kereta api]] juga akan dibangun menuju daerah lainnya seperti [[Kota Manado]]. Stasiun ini menjadi stasiun kereta api ke-23 di [[Sulawesi Selatan]].<ref name=stasiun>[http://www.sulselprov.go.id/berita-ekspose-penetapan-trase-jalur-kereta-api-makassarparepare.html Ekspose Penetapan Trase Jalur Kereta Api Makassar-Parepare] Diakses 21 Desember 2015</ref>
