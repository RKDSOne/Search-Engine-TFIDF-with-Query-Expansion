'''Halte Gunung Pandak''' merupakan perhentian kereta api penumpang di [[jalur kereta api Probolinggo-Paiton|jalur kereta non aktif api Probolinggo-Paiton]] dan berada di bawah pengawasan [[Daerah Operasi IX Jember]]. Halte ini berada di ketinggian +3 meter di atas permukaan laut dan berlokasi di sebelah timur Gunung Bentar, [[Gending, Probolinggo|Kecamatan Gending]]. Halte ini dinamai sesuai lokasi dimana terdapat sebuah gunung berpuncak pendek di belakang halte. Dalam [[bahasa Madura]], kata ''pandak'' berarti pendek.

Halte Gunung Pandak dibuat oleh perusahaan kereta api swasta Hindia Belanda yaitu Probolinggo Stoomtram Maatschappij (PbSM) yang mendapat konsesi izin pembangunan jalur kereta api di area Probolinggo dan Kraksaan. Halte ini tidak memiliki bangunan permanen dan berhenti sesekali jika ada penumpang yang naik dan turun di lokasi.
