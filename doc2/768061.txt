'''Stasiun Blimbing (BLM)''' adalah stasiun kereta api nonaktif yang berada di [[Blimbing, Gudo, Jombang]]. Stasiun ini merupakan salah satu dari beberapa stasiun kereta api yang berada di [[Jalur kereta api Kediri-Jombang]]. Dibangun pada tahun [[1897]] oleh [[Kediri Stoomtram Maatschappij]] (KSM), perusahaan kereta api [[Hindia Belanda]] yang beroperasi di daerah [[Kediri]]. Dulunya stasiun ini memiliki percabangan menuju Pabrik Gula Blimbing yang juga sudah nonaktif. <ref>[http://www.semboyan35.com/printthread.php?tid=6263 Misteri Jalur Mati di Daerah Kediri]</ref><ref>[http://www.semboyan35.com/printthread.php?tid=20&page=9 Jalur KA Kediri-Pare-Jombang]</ref>
