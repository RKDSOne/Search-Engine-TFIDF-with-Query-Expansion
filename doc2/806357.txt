Jalur kereta api Rancaekek-Tanjungsari
'''Jalur kereta api Rancaekek-Tanjungsari''' adalah salah satu jalur kereta api non-aktif di [[Jawa Barat]] terutama di [[Daerah Operasi II Bandung]]. Dulu jalur ini digunakan untuk mengangkut teh dan hasil bumi lainnya di daerah [[Sumedang]]. Pada masa pendudukan jepang (1942-1945), jalur ini dibongkar untuk pembangunan [[Jalur kereta api Saketi-Bayah]] yang banyak memakan korban jiwa. Sampai sekarang, bekas jalur, jembatan, dan yang lainnya masih ada meskipun banyak yang menjadi korban vandalisme.
