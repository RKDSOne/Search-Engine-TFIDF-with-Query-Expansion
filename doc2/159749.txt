'''Kereta api Malang Ekspres''' diluncurkan bulan September tahun 2006 sebagai alat transportasi alternatif tujuan [[Stasiun Surabaya Kota|Surabaya Kota]] -[[Stasiun Malang|Malang]] setelah adanya peristiwa [[semburan lumpur panas di Sidoarjo]]. Kereta ini melayani perjalanan dengan kelas bisnis dan dijalankan sekali sehari .

Kereta api ini berhenti di [[Stasiun Wonokromo]], [[Stasiun Sidoarjo|Sidoarjo]], [[Stasiun Bangil|Bangil]], [[Stasiun Lawang|Lawang]], [[Stasiun Singosari|Singosari]], dan [[Stasiun Blimbing|Blimbing]].

Sejak 1 Desember 2009, kereta api ini dihentikan pengoperasiannya karena okupansi penumpangnya di bawah rata-rata. Dan kini bekas rangkaiannya digunakan untuk rangkaian kereta api [[Kereta api Senja Kediri|Senja Kediri]] jurusan Malang-Pasarsenen lewat Semarang.
Sisa rangkaianya diletakkan di Dipo Malang.

Empat tahun setelah penghentian operasional kereta ini, PT KA Daop 8 Surabaya membuka lagi kereta ekspres rute Surabaya-Malang dengan nama Kereta Api [[Kereta api Penataran Ekspres|Penataran Ekspres]]. Kereta ini menggunakan rangkaian KA Penataran yang sudah diberi fasilitas pendingin udara (AC) dan hanya berhenti di stasiun-stasiun besar saja.
