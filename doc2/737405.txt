Adagio

[[File:Metronome Nikko.jpg|Gambar metronom mekanik, alat yang digunakan untuk menghitung tempo dalam musik seperti Adagio|thumb|right]]
'''''Adagio''''' (Bahasa Inggris, ''at ease'') adalah sebuah istilah dalam dunia [[musik]] yang berarti komposisi permainan musik dimainkan dengan [[tempo]] yang perlahan dan lembut.<ref name = "ref1">{{en}}{{citeweb|url=http://www.merriam-webster.com/dictionary/adagio|title=Adagio |accessdate=17 Mei 2014 |publisher=Merriam-webster}}</ref> Dalam dunia [[seni tari]], adagio berarti sebuah tarian [[ballet]] yang dilakukan oleh sepasang penari pria dan wanita atau 3 orang penari campuran untuk menampilkan tarian yang susah dan butuh keseimbangan karena melibatkan banyak gerakan pengangkatan penari dan putaran.<ref name = "ref1"/>

== Adagio dalam Dunia Musik ==
Dalam dunia musik, cepat-lambatnya tempo dapat diukur dengan alat yang disebut dengan [[metronom]].<ref name = "ref2">{{cite book|last=American Symphony Orchestra League|title=Journal of the Conductors' Guild, Vols. 1819|year=1998|publisher=The League|location=Viena|pages=27|url=http://books.google.cz/books?ei=kFELUKCBN_OW0QWu0_zpCg&hl=cs&id=zGcIAQAAMAAJ&dq=%22marcia+moderato%22+tempo+marking&q=%22marcia%22+}} {{ISSN|0734-1032}}</ref> Masing-masing nama kecepatan tempo menggunakan bahasa [[Italia]]. Berikut adalah urutan tempo dari yang lembat menjadi cepat.<ref name = "ref2"/>
* Larghissimo  sangat perlahan (19 BPM atau ke bawah)<ref name = "ref2"/>
* Grave  lambat dan serius  (2040 BPM)<ref name = "ref2"/>
* Lento  lambat (4045 BPM)<ref name = "ref2"/>
* Largo  melebar (4550 BPM)<ref name = "ref2"/>
* Larghetto  sedikit melebar (5055 BPM)<ref name = "ref2"/>
* Adagio  lambat dan lembut   (5565 BPM)<ref name = "ref2"/>
* Adagietto  sedikit lembut (6569 BPM)<ref name = "ref2"/>
* Andantino  sedikit lebih lembut dari andante  (7883 BPM)<ref name = "ref2"/>
* Andante  tempo seperti orang berjalan (8490 BPM) <ref name = "ref2"/>
* Andante moderato  kecepatan di antara  andante moderato (90100 BPM)<ref name = "ref2"/>
* Marcia moderato  kecepatan seperti orang yang sedang baris berbaris (8385 BPM)<ref name = "ref2"/>
* Moderato  dengan kecepatan sedang (100112 BPM)<ref name = "ref2"/>
* Allegro Moderato - cukup cepat (112-116)<ref name = "ref2"/>
* Allegretto  cepat hampir secepat allegro (116120 BPM)<ref name = "ref2"/>
* Allegro  cepat  (120160 BPM) <ref name = "ref2"/>
* Vivace  cepat dan hidup (132140 BPM)<ref name = "ref2"/>
* Vivacissimo  sangat cepat dan hidup (140150 BPM)<ref name = "ref2"/>
* Allegrissimo (or Allegro Vivace)  sangat cepat, melebihi Vivacissimo <ref name = "ref2"/>(168177 BPM)
* Presto  sangat cepat, melebihi Allegrissimo (180200 BPM)<ref name = "ref2"/>
* Prestissimo  paling cepat (200 BPM atau lebih)<ref name = "ref2"/>
