'''Stasiun Kedunggalar (KG)''' merupakan [[stasiun kereta api]] kelas III/kecil yang terletak di [[Kedunggalar, Kedunggalar, Ngawi]]. Stasiun yang terletak pada ketinggian +75 m ini termasuk dalam [[Daerah Operasi VII Madiun]]. Stasiun ini memiliki empat jalur dengan jalur 2 sebagai sepur lurus.

== Layanan kereta api ==
=== Kelas campuran (eksekutif-ekonomi AC) ===
* [[Kereta api Malioboro Ekspres|Malioboro Ekspres]], tujuan [[Stasiun Malang|Malang]] (jadwal pagi) atau [[Stasiun Yogyakarta|Yogyakarta]] (jadwal malam)

=== Kelas ekonomi AC ===
* [[Kereta api Gaya Baru Malam Selatan|Gaya Baru Malam]], tujuan [[Stasiun Surabaya Gubeng|Surabaya]]
* [[Kereta api Matarmaja|Matarmaja]], tujuan [[Stasiun Pasar Senen|Jakarta]] via [[Stasiun Semarang Tawang|Semarang]]
* [[Kereta api Logawa|Logawa]], tujuan [[Stasiun Surabaya Gubeng|Surabaya]] bersambung [[Stasiun Jember|Jember]]
* [[Kereta api Madiun Jaya Ekspres|Madiun Jaya]], tujuan [[Stasiun Madiun|Madiun]] dan tujuan [[Stasiun Yogyakarta|Yogyakarta]]
