'''Stasiun Rogojampi (RGP)''' merupakan [[stasiun kereta api]] kelas I yang terletak di [[Karangbendo, Rogojampi, Banyuwangi]]. Stasiun yang terletak pada ketinggian 89 m dpl ini berada di [[Daerah Operasi IX Jember]]. Stasiun ini memiliki tiga jalur dengan jalur 2 sebagai sepur lurus. Dahulu dari jalur 1 ke arah selatan, terdapat jalur cabang menuju [[Benculuk]]. Saat ini semua kereta milik [[Daerah Operasi IX Jember]] berhenti untuk menaikturunkan penumpang ketika berangkat dari dan ke [[Stasiun Banyuwangi Baru]]. 

==Layanan kereta api==
=== Kelas campuran (eksekutif-bisnis) ===
* [[Kereta api Mutiara Timur|Mutiara Timur]], tujuan[[Stasiun Surabaya Gubeng|Surabaya]]dan tujuan [[Stasiun Banyuwangi Baru|Banyuwangi]](khusus untuk jadwal malam bisa bersambung dari dan ke[[Kota Denpasar|Denpasar]],[[Bali]], menggunakan armada bus kerjasama antara Damri-PT KAI).

=== Kelas ekonomi AC ===
* [[Kereta api Sri Tanjung|Sri Tanjung]], tujuan [[Stasiun Surabaya Gubeng|Surabaya]] bersambung [[Stasiun Lempuyangan|Yogyakarta]] dan tujuan [[Stasiun Banyuwangi Baru|Banyuwangi]]
* [[Kereta api Tawang Alun|Tawang Alun]], tujuan [[Stasiun Bangil|Bangil]] bersambung [[Stasiun Malang Kotalama|Malang]] dan tujuan [[Stasiun Banyuwangi Baru|Banyuwangi]] 
* [[Kereta api Probowangi|Probowangi]], tujuan [[Stasiun Surabaya Kota|Surabaya]] dan tujuan [[Stasiun Banyuwangi Baru|Banyuwangi]]
* [[Kereta api Pandanwangi|Pandanwangi]], tujuan [[Stasiun Jember|Jember]] dan tujuan [[Stasiun Banyuwangi Baru|Banyuwangi]]
